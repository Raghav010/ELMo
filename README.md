# **How to run**
### **The News Classifier**
- `python agnewsclassifier.py`
- `BiLSTM.pth` needs to be in the same folder as `agnewsclassifier.py`
### **The ELMo Model**
- `python elmo10.py`

For both models to run `glove.6B.300d.txt` needs to be in the directory
